@Commands

ABS {num}

Description:

Sets the specified memory buffer block to its absolute value - if value is less than 0, will return the same value, but ..$*-1).

Arguments:

INT {num} - the specified memory block number to get the value from and assign to.

Example of usage:
SET 0 -32; ABS 0; SAY %^0;
Output:
32
